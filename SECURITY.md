# Security Policy

## Supported Versions

This is pre-release research software at this time. All security warnings are welcome on the head of the source tree. There are no bug bounties available for this software.

As this software moves into production settings, the list of supported versions on which security reports can be filed will be included here.

## Reporting a Vulnerability

There is no requirement for private disclosure at this time. If you find a vulnerability, please file an issue and describe it as a security bug.
