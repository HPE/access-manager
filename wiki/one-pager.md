# The Access Manager
The Access Manager provides unique capabilities for simple and secure access to data across multiple clouds, HPC data stores, relational databases and identity providers.
A key benefit of the simplicity is provable compliance (for both managers and mathematicians).

Design principles include:

* simplicity - controls are expressed simply to minimize the chance of errors
* portability – many kinds of data storage using any external identity control
* performance – the access manager does not degrade performance
* correct by default – users inherit correct configuration for global controls
* multi-tenant – non-cooperating organizations can selectively share data
